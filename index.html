Projects

Waterlogged

As an avid fly fisherman I set out to create an application to make the lives of fly anglers easier. Waterlogged is a web application that allows users to log past fly fishing trips and plan future fly fishing trips. Users can log details such as name of the body of water, date, season, gear used, how the water was accessed, and upload photos. This project was built using React.js, JSON Server for persistent data storage, and Tachyons for styling.
